# 📚 Anotações ADS - Base de Conhecimento

Repositório dedicado ao armazenamento, organização e evolução das anotações produzidas ao longo dos 6 semestres do curso de **Análise e Desenvolvimento de Sistemas**.

As anotações são estruturadas utilizando o [Obsidian](https://obsidian.md/), aproveitando sua flexibilidade com Markdown, links bidirecionais e organização em grafos de conhecimento.

---

## 🧠 Sobre o Projeto

Este repositório não é apenas um conjunto de anotações, mas sim uma **base de conhecimento em constante evolução**, construída com foco em:

- 📌 Fixação de conteúdo acadêmico  
- 📌 Organização de conceitos técnicos  
- 📌 Revisão contínua e aprendizado ativo  
- 📌 Criação de material de consulta rápida  
- 📌 Construção de um segundo cérebro digital  

---

## ✍️ Metodologia

As anotações seguem um processo estruturado:

1. 📖 Estudo do conteúdo (aulas, PDFs, livros)  
2. 🤖 Apoio de IA para estruturação e expansão  
3. ✍️ Escrita manual e adaptação da linguagem  
4. 🔁 Revisão contínua  
5. 🔗 Conexão entre tópicos (links internos)  

---

## 🧩 Estrutura do Repositório

